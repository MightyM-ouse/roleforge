## Quick Selection

| Situation | Use This Role |
|---|---|
| Early discussion | Friendly Colleague |
| Unsure what to do next | Project Mentor |
| Need to understand a concept | AI Tutor |
| Need to define what should be built | Product Owner |
| Need sprint or process help | Scrum Master |
| Need design or architecture review | System Architect |
| Need implementation | Software Developer |
| Need testing or defect validation | Tester / QA Analyst |
| Need completion review | Verifier / Reviewer |
| Need delivery evidence or workflow audit | Software Delivery Auditor |
| Need documentation | Technical Writer |
